## 信息

**Word排版 版式插件**是一个基于[玄武岩][basalt-link]项目的基础版式的附属版式插件。该插件修改了部分已有元素的样式，并通过引入 **word** 类和 **word-*** 类，添加了一系列新的样式，旨在符合其作者EasyEDG的审美风格。在该插件中，你几乎无需添加或仅需添加少量新样式即可直接使用。
